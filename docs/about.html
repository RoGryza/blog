About Me

Rodrigo Gryzinski, programmer currently living in Brazil. I've been working in finance for about 5 years now, currently I'm employed in my own company (Wuzu).

I've been programming for over a decade and a good chunk of my learning came from reading what others had to share online, so I set up this blog to pay it forward. My interests include:

  • Pure functional programming
  • Programming languages and compilers
  • Games and graphics programming
  • indieweb and the federated web
  • Free software and, more generally, free as in freedom creative work
